## Install

In server & client folder
```sh
npm install
```

## build-front

In client folder
```sh
npx webpack
```

## Usage

In server folder
```sh
npm start
```

## build-server

In server folder
```sh
npm build
```

## Run tests

In server folder
```sh
npm test
```
